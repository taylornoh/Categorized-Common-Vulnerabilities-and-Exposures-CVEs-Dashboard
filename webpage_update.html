About the Project

Description

Our project bridges the gap between common vulnerabilities and the public with a live dashboard on AWS. Users access recent vulnerabilities from tech giants, with real-time updates and solutions. AWS facilitates data collection via Amazon S3 and stores info in DynamoDB. As the core clouding platform, AWS is responsible for data storage, computing, and hosting our dashboard. Amazon DynamoDB serves as our primary database, storing a catalog of vulnerabilities reported by different companies. Amazon S3 is leveraged as the primary hosting solution for our static website which will fetch and display the data.
